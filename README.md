# hossein-amirkhani.github.io

Personal website of Hossein Amirkhani: <https://hossein-amirkhani.github.io>

Built with [Jekyll](https://jekyllrb.com/) and served via GitHub Pages. Based on the
[AcademicPages](https://github.com/academicpages/academicpages.github.io) template (a fork of
[Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes), MIT License).

Run locally:

```bash
bundle install
bundle exec jekyll serve
```
